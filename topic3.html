PORTFOLIO #3: Numbering Systems

Number Systems?

A number system is a way of representing and writing numbers using a set of symbols (digits) and rules. Different number systems use different bases (radix), which determine how numbers are formed and interpreted.

In computing and mathematics, number systems are essential for:

  • Counting and calculation
  • Storing data
  • Processing information in digital devices

Types of Number Systems

Number System Base (Radix) Digits Used
Decimal 10 0–9
Binary 2 0, 1
Octal 8 0–7
Hexadecimal 16 0–9, A–F

Uses and Significance of Each Number System

1. Decimal Number System (Base 10)

Digits: 0–9 | Example: 245₁₀

Uses: Everyday counting, money, time, distance, population, grades.

Significance: Most natural for humans; based on 10 fingers; used in daily records.

2. Binary Number System (Base 2)

Digits: 0 and 1 | Example: 1011₂

Uses: Computers, data storage, programming, networking.

Significance: Foundation of digital technology; matches electronic circuits (OFF/ON).

3. Octal Number System (Base 8)

Digits: 0–7 | Example: 157₈

Uses: Older systems, UNIX/Linux permissions.

Significance: Simplifies binary; each digit = 3 binary bits.

4. Hexadecimal Number System (Base 16)

Digits: 0–9, A–F | Example: 2F₁₆

Uses: Memory addresses, programming, debugging, web color codes.

Significance: Compact form of binary; easier to read than long binary values.

ANALYSIS & REFLECTION

Before learning about number systems, I only thought of numbers as the ones we use every day, like when we count money, check time, or solve math problems in school. I did not realize that there are actually different types of number systems and that each one has its own purpose. After studying the decimal, binary, octal, and hexadecimal number systems, I learned that numbers are not only for people but also very important for computers and technology.

The decimal number system is the one I am most familiar with because it is what we use in our daily lives. It is based on the digits 0 to 9, and it makes counting and solving problems easier for humans. I realized that this system is designed for us since it matches how we naturally count.

What really amazed me was the binary number system. It only uses two digits, 0 and 1, yet it is the main language of computers. All apps, games, and messages are built from just 0s and 1s.

The octal and hexadecimal number systems were new to me. I learned they make binary numbers easier to read. Hexadecimal is also used in web colors and programming.

Overall, learning about number systems helped me see math in a different way. I realized numbers are the foundation of computers and digital devices.
